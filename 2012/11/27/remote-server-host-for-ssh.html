SSH反向链接配置

应用场景:
A为公网上的一个SSH服务器, 端口为9944, ip为9.9.9.9, 用户名为a, 本地开放端口为10000,
B为内网里的一个SSH服务器, 端口为22, 用户为b.
如今A向B发起SSH请求,实行远程操控.

A的SSH服务端口为9944.
B的SSH服务端口为22.

  • 先在B上执行: B $ ssh -R 10000:localhost:22 a@9.9.9.9 -p 9944
  • 之后就可以在A上执行: A $ ssh b@localhost -p 10000

enjoy it.
